Detailed Description

Calibration class for monocular cameras::

    images = [cv2.imread("mono%d.png") for i in range(8)]
    mc = MonoCalibrator()
    mc.cal(images)
    print mc.as_message()

calibrated image are valid) to 1 (zoomed out, all pixels in
